# flek_select Example

This example app demonstrates the usage of the `flek_select` package.

## Running the Example

```bash
cd example
flutter pub get
flutter run
```

## Features Demonstrated

### Select Tab

1. **Basic Select** - Default behavior using GestureDetector
2. **Custom Input Builder + Tappable** - Using `inputBuilder` with the `tappable` package for touch feedback
3. **Custom Input Builder + InkWell** - Using `inputBuilder` with Material Design ripple effect

### ToggleButtonGroup Tab

1. **Basic ToggleButtonGroup** - Default behavior using GestureDetector
2. **Custom Button Builder + Tappable** - Using `buttonBuilder` with the `tappable` package
3. **Custom Button Builder + InkWell** - Using `buttonBuilder` with Material Design ripple effect
